# Java Solitaire
A game of Solitaire written in Java. Uses Stacks for Tableaus, Foundations, and Deck. Uses Swing for GUI.

## Features:
  * Classic Klondike Solitaire.
  * Ability to deal 1 or 3 cards from the deck.
